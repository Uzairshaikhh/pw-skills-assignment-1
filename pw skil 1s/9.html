1. <strong> and <em>
    • Use <strong> to indicate strong importance, typically rendered as bold.
    • Use <em> to emphasize text, typically rendered as italic.
    • Best practice: Use <strong> for emphasis that is stronger than <em>.
  2. <u>
    • Defines text that should be stylistically offset from normal prose, typically rendered as underlined.
    • Use sparingly as it can affect readability and be associated with hyperlinks.
    • Best practice: Avoid using <u> for anything other than hyperlinks.
  3. <del> and <ins>
    • Use <del> to represent deleted text, typically rendered with a strikethrough.
    • Use <ins> to represent inserted text, typically rendered with an underline.
    • Best practice: Use these tags within <blockquote> or <q> to denote changes in quoted text.
  4. <sup> and <sub>
    • Use <sup> to represent superscript text, typically used for footnotes, mathematical exponents, etc.
    • Use <sub> to represent subscript text, typically used for chemical formulas, mathematical subscripts, etc.
    • Best practice: Use these tags sparingly and only when necessary for clarity.
