MEAN Stack
A full stack JavaScript framework consisting of MongoDB, Express.js, AngularJS, and Node.js, used for building dynamic web applications.
MERN Stack
Similar to the MEAN stack, but with React.js replacing AngularJS as the frontend framework.
LAMP Stack
Short for Linux, Apache, MySQL, and PHP, it's a traditional stack used for building dynamic websites and web applications.
Python Django Stack
A full stack framework utilizing Python and Django for backend development, providing robust features for rapid development and scalability.
Ruby on Rails
A full stack web application framework written in Ruby, optimized for programmer productivity and convention over configuration.
MEVN Stack
Similar to the MEAN stack, but with Vue.js replacing AngularJS as the frontend framework.
Serverless Stack
A modern approach where applications are built using serverless architectures, abstracting away server management and scaling concerns.
