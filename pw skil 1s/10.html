1. HTML
    Tech Stack Name Primary Use Cases Key Features/Benefits
    HTML (Hypertext Markup Language)
    • Creating the structure and content of web pages.
    • Defining the layout and organization of elements.
    • Providing semantic meaning to content.
    • Standard markup language for creating web pages and web applications.
    • Supports multimedia, forms, and other dynamic content.
    • Ensures accessibility and search engine optimization (SEO).
  2. CSS
    Tech Stack Name Primary Use Cases Key Features/Benefits
    CSS (Cascading Style Sheets)
    • Styling the appearance of HTML elements.
    • Creating layouts, colors, fonts, and animations.
    • Implementing responsive design for various screen sizes.
    • Separates content from presentation, improving maintainability.
    • Offers powerful selectors and inheritance for efficient styling.
    • Facilitates consistent branding and user experience across a website.
  3. JavaScript (JS)
    Tech Stack Name Primary Use Cases Key Features/Benefits
    JavaScript (JS)
    • Adding interactivity and behavior to web pages.
    • Manipulating HTML and CSS to dynamically update content.
    • Handling user input, events, and form validation.
    • Client-side scripting language for web development.
    • Supports asynchronous programming for responsive and dynamic applications.
    • Large ecosystem of libraries and frameworks for various tasks (e.g., React, Angular, Vue).
